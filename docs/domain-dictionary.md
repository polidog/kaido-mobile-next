# Domain Dictionary: kaido-mobile-next

> 最終更新: 2026-07-10

## 地図表示 (map)

| 用語 | 定義 | 英語名 / 別名 |
|------|------|--------------|
| マーカー | 地図上に表示されるポイントのピンアイコン | Marker |
| Polyline | 地図上に描画される連続線分。街道ルートの表示に使用 | Polyline |
| ビューポート | 現在画面に表示されている地図の範囲 | VisibleRegion |
| 情報ウィンドウ | マーカータップ時に表示されるタイトル・カテゴリのポップアップ | InfoWindow |
| カメラ | 地図の視点。位置（緯度経度）・ズームレベル・回転角度を持つ | CameraPosition |

## ナビゲーション (navigation)

| 用語 | 定義 | 英語名 / 別名 |
|------|------|--------------|
| GPS追従 | 端末のGPS位置にカメラを追従させるモード | Location Tracking |
| コンパスモード | 端末の方位に合わせてカメラを回転させるモード | Compass Mode / bearing |

## コンテンツ (content)

| 用語 | 定義 | 英語名 / 別名 |
|------|------|--------------|
| ポイント | 地図上に表示される地点情報。id・タイトル・緯度経度・説明・カテゴリ・画像を持つ | Point / Spot |
| 宿場 | 街道沿いの宿泊・休憩施設があった町 | shukuba |
| 名所 | 歴史的に著名な場所・観光スポット | meisho |
| 一里塚 | 一里（約4km）ごとに設置された道標の塚 | ichirizuka |
| 浮世絵 | 浮世絵に描かれた場所。主に東海道・中山道に存在 | ukiyoe |
| 見付 | 宿場の入口にあった見張り番所 | mitsuke |
| 本道 | 街道の正規ルート | main route / distance |
| 寄り道 | 本道から外れた観光・歴史的ルート | detour |

## データ同期 (data-sync)

| 用語 | 定義 | 英語名 / 別名 |
|------|------|--------------|
| バンドルJSON | アプリビルド時に同梱される初期データファイル | bundled assets |
| ファイルキャッシュ | APIから取得したデータをローカルファイルに保存したもの | file cache |
| API コンテキスト | どの街道のデータを取得するかを示す識別子（tokaido, nakasendo 等） | context / aliasName |

## ユーザーサポート (user-support)

| 用語 | 定義 | 英語名 / 別名 |
|------|------|--------------|
| 問い合わせ | ユーザーがメールで運営に連絡する機能 | Contact |
| 位置情報選択 | 問い合わせ時に地図上で該当箇所を指定する機能 | ContactMap |

## アプリ構成 (app-config)

| 用語 | 定義 | 英語名 / 別名 |
|------|------|--------------|
| 街道 | 江戸時代に整備された主要道路。五街道は東海道・中山道・甲州道中・日光道中・奥州道中 | kaido |
| KaidoConfig | アプリ固有の設定オブジェクト。アプリ名・APIコンテキスト・テーマ色を含む | KaidoConfig |
| スプラッシュ | アプリ起動時に表示される画面 | SplashPage |
